# 🎸 Catálogo de Bandas

## Tema
Catálogo de bandas de rock e metal dos anos 80 até hoje.

## Descrição
Aplicação desenvolvida com Vue 3, Vuetify e Vue Router para cadastrar,
listar, editar e excluir bandas musicais. Os dados são mantidos em memória
usando um store compartilhado com ref() do Vue.

## Como rodar

```bash
npm install
npm run dev
```

## Tecnologias utilizadas
- Vue 3
- Vuetify
- Vue Router
- Vite
